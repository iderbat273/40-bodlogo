Prim's MST 
Бодлогын өгүүлбэр
Нэгэн чиглэлгүй граф өгөгдсөн. Энэ графын хамгийн бага оновчтой бүрхүүл (Minimum Spanning Tree - MST)-ийг олж, түүний ирмэгүүдийн нийт жинг бодно.

Оролт
Эхний мөр: Хоёр бүхэл тоо n ба m, хаана n нь оройн тоо, m нь ирмэгүүдийн тоо.

Дараагийн m мөр: Гурван бүхэл тоо u, v, ба w, хаана u ба v нь ирмэгээр холбогдсон оройнууд, w нь тухайн ирмэгийн жин.

Сүүлийн мөр: Эхлэх оройн дугаар s.

Гаралт
Хамгийн бага оновчтой бүрхүүлийн нийт жинг хэвлэ.

Хязгаарлалт
1 ≤ n ≤ 3000

1 ≤ m ≤ n*(n-1)/2

1 ≤ w ≤ 10^5

1 ≤ s ≤ n
