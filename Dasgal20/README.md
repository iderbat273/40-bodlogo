Crab Graphs 
Бодлогын өгүүлбэр
Нэгэн чиглэлгүй граф өгөгдсөн. "Crab" гэдэг нь нэг "толгой" орой болон түүнтэй холбогдсон хамгийн ихдээ t "хөл" оройтой дэд граф юм. Бидний зорилго бол графаас сонгон авч болох хамгийн олон оройтой crab-уудын цуглуулгыг олох юм. Нэг орой нь нэгээс олон crab-д харьяалагдах боломжгүй.

Оролт
Эхний мөр: Туршилтын тоо c

Туршилт бүрийн хувьд:

Эхний мөр: Гурван бүхэл тоо n (оройн тоо), t (хөлний дээд тоо), m (ирмэгийн тоо)

Дараагийн m мөр: Ирмэгүүдийг илэрхийлсэн u ба v оройнууд

Гаралт
Туршилт бүрийн хувьд олж болох хамгийн их crab-уудын нийт оройн тоог хэвлэ.

Хязгаарлалт
1 ≤ n ≤ 100

1 ≤ t ≤ 100

1 ≤ m ≤ n*(n-1)/2
